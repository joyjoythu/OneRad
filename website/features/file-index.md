# @ 文件索引

项目文件多了以后，「那个包含 LASSO 结果的报告在哪」会变成高频问题。@ 文件索引让 Agent 能快速定位项目里的任何文件。

## 用法

在项目聊天输入框中输入 **`@`**，触发文件索引面板。Agent 会扫描整个项目目录，建立可搜索的文件索引。

之后你可以直接用自然语言问文件位置：

> 「那个包含 LASSO 结果的报告在哪」

> 「把五折交叉验证的 ROC 图找出来」

Agent 基于索引快速定位到具体文件路径，不用自己一层层翻目录。

## 典型场景

| 场景 | 问法示例 |
|------|---------|
| 找分析产物 | 「最新的分析结果在哪个目录」 |
| 找特征文件 | 「提取的特征 CSV 在哪」 |
| 找报告 | 「生成的 Word 报告路径」 |
| 找配置 | 「当前用的 YAML 参数文件是哪个」 |

::: tip
分析产物都在项目目录的 `output/` 下，Agent 对产物结构了如指掌——直接问它比手动翻目录快得多。
:::
