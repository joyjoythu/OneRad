# 快捷操作

对话输入框左侧的 **⚡ 快捷操作** 下拉菜单，把最常用的指令做成一键入口，省去打字。

## 菜单项一览

| 菜单项 | 行为 | 说明 |
|--------|------|------|
| **开始分析** | 填入提示词 | 检查项目配置与数据完整性，开始执行完整影像组学分析流程 |
| **重新分析** | 填入提示词 | 基于当前项目配置重新执行完整分析——Agent 会先检查现有产出物，说明将复用哪些数据与参数、会覆盖或新生成哪些结果，再逐步执行 |
| **格式转换** | 填入提示词 | 扫描项目目录中的 DICOM 数据，全部转换为 NIfTI（`.nii.gz`），转换前先告知输入目录和计划的输出目录 |
| **导出对话（Markdown）** | 立即执行 | 把当前对话导出为 `.md` 文件 |
| **导出对话（Word）** | 立即执行 | 把当前对话导出为 `.docx` 文件 |
| **清除当前任务** | 弹窗确认 | 清除当前任务上下文并开始新对话，**历史对话仍会保留** |

## 两种触发方式

快捷操作分两类，注意区别：

**填入提示词类**（开始分析 / 重新分析 / 格式转换）：点击后，预设提示词**填入输入框但不直接发送**——你可以先修改补充（比如调整输出目录、追加额外要求），确认无误后手动发送。如果你输入框里已有内容，提示词会追加到末尾。

**立即执行类**（导出对话 / 清除任务）：点击即执行。导出对话只读取会话快照，**Agent 运行中也能用**；清除任务会先弹确认框防误触。

## 典型用法

- **评委 / 新用户**：新建项目后直接点「开始分析」，不用想第一句话怎么说
- **增量实验**：数据或参数变更后点「重新分析」，Agent 会明确告诉你哪些产物被复用、哪些会更新
- **数据预处理**：拿到 DICOM 原始数据先点「格式转换」，再点「开始分析」
- **留档**：做完一轮分析点「导出对话（Word）」，把完整的分析决策过程存档
