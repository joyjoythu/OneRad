# Todo List 步骤展示

对话界面右侧的 **Todo 面板**是 Agent 的「进度公告栏」：它把当前任务拆解成清晰的步骤列表，实时更新执行状态，让你随时知道 Agent 做到哪一步、接下来要做什么。

## 三种状态

| 状态 | 含义 |
|------|------|
| ⬜ **pending** | 待执行的步骤 |
| 🔵 **in_progress** | 正在执行的步骤（同时至多一个） |
| ✅ **completed** | 已完成的步骤 |

## 什么时候会出现

当 Agent 接到多步骤任务时（如「开始分析」），它会先规划步骤并写入 Todo 面板，例如：

```
✅ 探索项目结构
✅ 配对影像与掩膜
🔵 提取影像组学特征
⬜ 临床表匹配与审查
⬜ LASSO 建模与交叉验证
⬜ 生成统计报告
```

每完成一步，面板实时更新。配合[逐例提取进度条](/guide/workflow#环节四-特征提取)，粗粒度（步骤）和细粒度（病例）进度都一目了然。

## 设计细节

- **全量替换**：Agent 每次更新都是重写整个列表，保证状态一致，不会出现「僵尸步骤」
- **与对话联动**：步骤状态变化通过 SSE 实时推送，刷新页面后也能从存档恢复显示
- **只读展示**：面板仅供查看，步骤的规划与推进由 Agent 负责；想调整顺序或跳过某步，直接在对话里说即可
