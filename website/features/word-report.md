# Word 统计报告

分析完成后，OneRad 自动生成 Word 统计报告（`AutoRadiomics_Report.docx`），把方法描述、统计结果和图表汇总成一份文档，省去手动整理。

## 报告内容

| 部分 | 内容 |
|------|------|
| 方法学描述 | 特征提取参数、LASSO + 逻辑回归建模流程、交叉验证设置 |
| 结果表格 | AUC、敏感度、特异度等逐折及汇总指标 |
| 图表 | ROC 曲线、校准曲线、DCA 决策曲线、SHAP 特征贡献图 |
| 结果解读 | 三段式中文解读（见下） |

## 自动结果解读

分析成功后，Agent 自动调用结果解读，向报告注入三段中文分析：

1. **【模型性能解读】** — AUC / 敏感度 / 特异度 / 逐折稳定性
2. **【特征意义解读】** — 系数 / OR 值 / SHAP 方向一致性
3. **【SHAP 可解释性解读】** — top 特征的贡献方向与幅度

解读是**幂等**的：重复触发不会叠加重复小节。

## 排版格式

报告自动套用规范的中文文档格式：

| 元素 | 格式 |
|------|------|
| 正文 | 中文宋体、西文 / 数字 Times New Roman、小四（12pt）、1.5 倍行距 |
| 标题 | 黑体三号（16pt）加粗居中 |
| 一级标题 | 黑体四号（14pt）加粗（显式覆盖 Word 默认蓝色主题） |
| 二级标题 | 黑体小四（12pt）加粗 |
| 表格 | 五号（10.5pt），表头加粗居中 |
| 图片 | 段落居中 |

## 格式不满意？一句话重排

如果报告格式混乱或你想重新排版，直接说：

> **重排报告**

Agent 会自动定位项目内最新的分析输出，把报告重排为上述格式，**原地保存**（不生成备份文件，可反复执行）。

## 用 Word 工具写任意文档

除了统计报告，Agent 还能按同样的格式创建和追加任意 Word 文档。例如：

> 「帮我把今天的实验记录写成一个 Word 文档」

> 「在报告里追加一段局限性讨论」

内容用 Markdown 组织（`#` 各级标题、`-` 列表、`**粗体**`），Agent 会转换为规范格式。新建文档不会覆盖已有同名文件。
